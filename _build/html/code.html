patternchecker.check_for_dates(text)[source]¶

Removes dates and numbers that may look like dates from a string.

Parameters:text (str) – string to remove dates from
Returns:list of dates that were removed from string str: original string with dates removed
Return type:list 
>>> dates,non_dates = check_for_dates('2/12/2015, 2/2015, boxing')
>>> dates == ['2/12/2015', '2/2015']
True
>>> non_dates == ' boxing'
True

Parses string and categorzes each words as allowed/notallowed/indeterm.

True
